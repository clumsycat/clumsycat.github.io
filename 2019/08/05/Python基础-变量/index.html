Python中is和==的区别

  • is用于比较两个变量是否引用了同一个内存地址 is 表示的是对象标示符（object identity），作用是用来检查对象的标示符是否一致，也就是比较两个对象在内存中的地址是否一样。
  • ==用于比较两个变量的值是否相等,== 表示的是相等（equality），是用来检查两个对象是否相等。

is比==更严格的检查

Python类中有哪些成员，如何分类他们，他们又有哪些用途

  • 变量
    • 类变量：整个类通用的变量
    • 成员变量： 用于定义类实例中的参数类型和默认值，通常再init中由self.定义
  • 函数
    • 成员函数：参数第一项永远为self，可以访问类变量，成员变量，函数局部变量
    • 类函数：@classmethod修饰，用于访问类变量和成员变量，以及对这些变量进行修改
    • 静态函数：@staticmethod修饰，纯静态，不可访问和修改任意变量
    • 普通函数：无self作为参数，不可访问类变量，成员变量，只可以做和实例各参数无关的处理

Python中列表，元组的相同之处及区别都有哪些？集合与字典呢？

元组和列表相同点：可以遍历，索引，切片，连接（join），元素可以是任意类型

元组：(),不可以更改，不可以排序
列表：[],可以增删改查，可以排序

集合和字典相同点：无序，没有重复项（字典是么有重复的key），可以增删改，可以遍历

集合：集合中的元素可以是任意类型，集合间有交并差集运算
字典：{key:value},成对出现，key只能是字符串，数字或元组，value可以是任意数据类型

Python中的模块和包是什么，如何自定义并使用？

一个模块即为一个python文件，通过import文件名即可引用模块到当前python文件中，可以使用模块中定义的类或者函数

一个包即为包含一个或多个模块的文件夹，其中必有init.py(通常为空)，通过引用包可以自由调用包中的任一模块中的类或函数。
